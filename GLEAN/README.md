# GLEAN

Group Level Exploratory Analysis of Networks                              
Adam Baker 2015

For details on how to set up a new GLEAN analysis type:
```matlab
help glean_setup
```
For details on particular GLEAN settings type:
```matlab
help glean_check
```
To run a GLEAN analysis type:
```matlab
run_glean(GLEAN)
```
